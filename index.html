Descripción del sistema

El sistema de Huaraches Los Osos funciona como un servicio de alimentos donde intervienen procesos de atención al cliente, preparación de alimentos y entrega del pedido.

Proceso de venta

El proceso de venta en Huaraches Los Osos inicia cuando el cliente llega al establecimiento o realizar un pedido para llevar. El cliente consulta el menú disponible y selecciona el platillo que desea ordenar. Posteriormente, el pedido es comunicado al personal de cocina, quienes se encargan de la preparación del platillo utilizando los ingredientes disponibles. Durante este proceso se busca mantener la calidad, presentación y sabor característicos del negocio. Una vez preparado el alimento, este es entregado al cliente en su mesa si el consumo es dentro del establecimiento, o bien se prepara para llevar. Finalmente, cuando el cliente termina de consumir su alimento o recibe su pedido para llevar, se realiza el cobro correspondiente en caja. El pago puede realizarse en efectivo o mediante otros medios disponibles. Después del pago, se concluye la venta y el cliente puede retirarse del establecimiento.
